# vcf-compare

Helper functions for plotting comparisons of VCF files.

For examples in a Jupyter notebook see: [example](example/).

## Plots implemented

- [x] 2-way Venn diagram of shared and unique variants
- [x] 4-way Venn diagram of shared and unique pass and filtered variants
- [/] Euler diagram of sets of shared and unique pass and filtered variants 
- [ ] Variant position plots
- [ ] Variant metric range plots
